# 红黑卧底游戏

基于 Node.js + Express + Socket.IO 的在线队伍身份分配器。玩家加入后只负责选择队伍；主控每局点击开始时，系统在各队伍内随机分配卧底（队伍不变）。

## 在线地址
- 玩家入口：https://game-f8jw.onrender.com/
- 主控入口：https://game-f8jw.onrender.com/admin.html

> Render 免费实例在空闲时会休眠，唤醒时首个请求可能变慢。

## 登录与房间
- 主控密码：bmsclub
- 主控先登录并设置房间码（房间码你们自己决定）
- 玩家加入时必须输入房间码

## 快速开始
1. 主控在主控入口输入房间码 + 密码登录
2. 玩家在玩家入口输入房间码/名字/红队或黑队加入
3. 主控点击“开始游戏”，每局会重新刷新身份（显示刷新时间）

## 游戏规则（默认逻辑）
- 分红队、黑队；玩家队伍在加入时确认，之后不变
- 主控每局点击开始后，系统在每个队伍内随机 1–2 名卧底
- 卧底人数：
  - 队伍人数 ≤ 1：不分配卧底
  - 队伍人数 = 2：固定 1 名卧底
  - 队伍人数 ≥ 3：随机 1 或 2 名卧底
- 身份显示：
  - 普通玩家：只看到“普通队员”
  - 卧底：看到“己方卧底 + 对方卧底是谁”
- 主控可以踢人

## 本地运行（可选）
```bash
npm install
npm start
```
